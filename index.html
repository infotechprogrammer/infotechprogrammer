About Me

I am a creative individual passionate about Web Development and Content Creation.

I can make nice and attractive static front end websites using HTML, CSS and Javascript. Mostly worked with Bootstrap framework and templates to provide a versatile look to the webpage. If anyone wants a nice and good looking static website for their startup can DM me on Instagram for that.

Back End Developer

Specialised in PHP for making dynamic websites like admin panels for controlling backend operations along with using MySQL database to store data.
